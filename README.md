# Pedra-papel-tesoura

### Descrição 
> Jogo do pedra, papel e tesoura feito em PHP+HTML+CSS+MySQL. Onde ao jogar contra o CPU os dados vão ser armazenados no banco de dados e ao final o usuário poderá consultar suas estatísticas.
